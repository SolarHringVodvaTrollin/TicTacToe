Administration manual

Instructions for installing and running the program on clients computer.

---------- 1. The code is written in Java (v.1.8) so in order to run the application you need to have java jdk installed. If haven't got it already you follow the links below:

- Download Java JDK 8

- Instructions for installing Java JDK

  1. Download the .zip and unpack it on your computer

    • Direct link to TicTacToe.zip
  2. In order to run the program you need a terminal. This can be Powershell for Windows or any bash shell for Unix.
  • To install the program you need to run the script
    `   bin/install` for Unix  


    `   bin/install.bat` for Windows
  • The project will be installed into the folder ~/Games for Unix and C:\Games for Windows.
